Things We Did Not Cover

While this tutorial is meant to give you the taste of what can be done with Orange Scripting, the things you have learned may be sufficient, for instance, to try out and implement some ideas you have for machine learning and data mining. However, we should warn you that Orange contains many other things that perhaps we did not even mention. You should browse through Orange’s documentation to find more.
